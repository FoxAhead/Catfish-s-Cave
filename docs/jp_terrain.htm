Terrain

If there's one thing I couldn't stand about Test of Time, it was the default graphics. They are actually worse than the 8-bit graphics of the classic game. The most visible graphics in the game are the terrain tiles and the default ones had to go.

24-bit Terrain Collection

This is a compilation of 24-bit terrain graphics for Test of Time. Most are made or modified by me, but some are created by others (see the Readme file in the download for full credit details). Some have been released elsewhere, but many of those have been modified since for better tiling. The screenshots below show a few examples. Resource icons are included in the download.
