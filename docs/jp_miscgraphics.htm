Miscellaneous Graphics

Graphics Mod Pack for the Original Game*

For those people who think that Test of Time's default graphics are too dark, indistinct and well, just plain ugly (like I do), here's an alternative. The pack contains many of my previously released 24-bit colour terrain graphics, as well as animated terrain resources and team colour masks (thanks to Mercator's CivSprite and SpriteGen utilities), plus units and cities art by Fairline and Curt Sibling.

Download 1.03 MB (Last modified 28/02/2020. Added @COSMIC2 movement multiplier entries to the rules files to fix a long-standing ToTPP bug that allowed unlimited movement on roads and rivers, and the same for alpine units.)

* Requires at least v0.9 of the Test of Time Patch Project.


Graphics Mod Pack for the ToT Conversion of Bernd Brosing's Imperium Romanum Scenario*

This is an unofficial graphics modification for the Test of Time version of Bernd Brosing's classic Imperium Romanum scenario. In 2006, prolific Civ2 icon artist Fairline created a set of units for the converted scenario and posted his own graphics modpack in the old Scenario League forum. I'd played this scenario some years earlier using a different terrain set and this download was a mixture of those early graphics, Fairline's units, Curt Sibling's cities and people, and a more recent selection of terrain graphics. In 2013, Fairline released a second graphics modpack for the scenario. It included new units, cities and people. With his permission, this pack has been updated with these graphics. This modpack requires the original ToT conversion.

Download 913 kB (Last modified 19/09/2014. ToTPP v0.9 version. Includes 8-frame combat animation from Icons.bmp and additional Terrain2.bmp overlays.)

* Requires at least v0.9 of the Test of Time Patch Project.


Graphics Mod Pack for Techumseh's Operation Market Garden Scenario*

This is a graphics modification for Techumseh's World War II scenario about the Allies' failed plan to secure bridges in the Netherlands as far north as the Rhine in late-1944. It remains today the largest airborne operation in history. I played this scenario back in 2004 and, as I almost always do, ended up changing the graphics at the same time, particularly the terrain and cities. The modification retains most of the original units, but some have been replaced with more recent icons. This pack is unofficial; it was only ever intended for personal use, but over the years there have been some requests for it - so here it is. The scenario suffers from the sprite problem that seems to plague ToT scenarios. This is fixed in the pack. You can download the original scenario in its entirety here.

Download 377 kB (Last modified 28/02/2020. Added @COSMIC2 movement multiplier entries to the rules file to fix a long-standing ToTPP bug that allowed unlimited movement on roads and rivers, and the same for alpine units. Added more recent versions of Fairline's unit graphics.)

* Requires the Test of Time Patch Project.


Rusted Skin Set for Test of Time

This is a set of replacement graphics for Test of Time's User Interface. The set I was using previously was starting to get a bit old, so I thought it was time for a change. The pack replaces the game's trim, buttons, checkboxes and background textures.
